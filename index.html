Customer Segmentation

In this project, customers' demographical data of an E-commerce online shop has been analyzed in order to perform Customer Segmentation by unsupervised learning including Hierarchical Clustering, K-Means Clustering, and PCA etc.

Predicting Customer Purchase Incidence

By applying logistics regression, a model for predicting a customer will purchase or not has been established. Based on that, a price elasticity analysis has been performed in order to examine the response to price changes for each customer group.

Predicting Customer Purchase Revenue

By applying liner regression, a model for forecasting customer purchase value for the coming year has been established and evaluated by RMSE, MAE, and Pearsonr.

Predicting Customer Churn

As we know acquiring a new customer is much more expensive than keeping an existed customer, so in this project, a logistic regression model has been established to predict a customer will churn or not while using tree based feature selection method.

A/B Testing Marketing Campaigns

This project aims to compare three campaigns by statistical metrics in order to decide which one has the best effect, then the chosen one will be launched on a large scale.

Examining Spending Habit with Age

By applying linear regression, decision tree, and random forest models, this project tries to figure out how spending behavior changes with age as well as how well each model can capture that changes.
